# TaskFlow — AppDev1 Midterm Exam

A lightweight personal task manager built with Bootsrap 5, Angular v19 and Angular21 since it is both similar.
Because of the Technical requiremnts, I used Angular v19 as my system's Node.js version was incompatible with Angular v21's requirement of Node v22.12+, but I've upgrade my node version angular v21.

## Features
- Add, edit, delete tasks
- Toggle task status (Pending → In Progress → Completed)
- View task details with child route tabs (Info / Edit)
- Parameterized routing with `/tasks/:id`
- Bootstrap 5 responsive UI with colored priority and status badges

## Tech Stack
- Angular v19 and upgraded to Angular v21 (standalone components)
- TypeScript interfaces and union types
- Angular Services for all CRUD
- Bootstrap 5
- Deployed on Netlify
